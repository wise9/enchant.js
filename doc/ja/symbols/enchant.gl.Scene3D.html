Class Detail

enchant.gl.Scene3D()
表示Sprite3Dツリーのルートになるクラス. 現在, 複数定義することは出来ず, 最初に定義したScene3Dが返される. 
  var scene = new Scene3D();
  var sprite = new Sprite3D();
  scene.addChild(sprite);

Field Detail

{Number[]} backgroundColor
Scene3Dの背景色
{enchant.gl.Sprite3D[]} childNodes
子要素の配列. このシーンに子として追加されているSprite3Dの一覧を取得できる. 子を追加したり削除したりする場合には, この配列を直接操作せずに, enchant.gl.Scene3D#addChildやenchant.gl.Scene3D#removeChildを利用する.
{enchant.gl.PointLight[]} lights
照明の配列. 現在, シーンに適用される光源は0番目のみ. このシーンに追加されている光源の一覧を取得する. 照明を追加したり削除したりする場合には, この配列を直接操作せずに, enchant.gl.Scene3D#addLightやenchant.gl.Scene3D#removeLightを利用する.
