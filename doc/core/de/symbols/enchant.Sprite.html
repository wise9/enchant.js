Field Detail

{Number|Array} frame
Die Indizes der darzustellenden Frames. Die Frames mit der selben Größe wie das Sprite werden aus der enchant.Sprite#image image Variable, beginnend an der oberen linken Ecke, angeordnet. Wenn eine Nummbersequenz übergeben wird, wird der dargestellte Frame automatisch gewechselt. Am ende des Arrays der Sequenz wird diese neugestartet. Wenn ein Wert in der Sequenz auf null gesetzt wird, wird das automatische Framewechseln gestoppt. 
var sprite = new Sprite(32, 32);
sprite.frame = [0, 1, 0, 2]
//-> 0, 1, 0, 2, 0, 1, 0, 2,..
sprite.frame = [0, 1, 0, 2, null]
//-> 0, 1, 0, 2, (2, 2,.. :stop)
